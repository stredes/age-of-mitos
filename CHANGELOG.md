# Changelog - Age of Mitos

## v0.2.0 - Multi-PC Improvement Sprint (2026-07-16)

### Features Added
- **AI System Improvements**
  - Director: personality types (aggressive, defensive, balanced)
  - Economy: better resource balancing, market trading
  - Military: varied attack formations, retreat logic
  - Builder: smarter placement, defense structures

- **Combat System Improvements**
  - Area damage and multi-target support
  - Terrain bonuses for damage calculation
  - Different projectile types (arrow, bolt, fireball, boulder)

- **UI/UX Improvements**
  - HUD: pill-shaped resource backgrounds, flash on change
  - Build menu: building previews, descriptions
  - Train menu: unit previews, queue display
  - Selection panel: better unit/building info
  - Tooltip system for all buttons

- **World System Improvements**
  - Better terrain variety in world generation
  - Improved resource node placement
  - Smoother fog of war reveal/hide
  - Enhanced weather system effects

- **Animation Improvements**
  - Smoother unit animation transitions
  - Better building construction stages
  - More ambient life in decorative animations
  - Improved combat and destruction particle effects
